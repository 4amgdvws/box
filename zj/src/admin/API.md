# API接口说明文档

RegUser.php 

本接口仅支持`post`方法。
```
参数：
    username 用户名
    password 密码

```

```
返回数据:
 - 成功 1
 - 失败 0 
 - error_msg 错误信息
{
    "state": 0,
    "error_msg": ""   
}
```
